# 娘家保健生技測試網站

模擬娘家保健生技官方網站的靜態測試網站，用於 UXAgent 自動化測試。

## 快速啟動

```bash
python3 -m http.server 3000
```

然後在瀏覽器開啟：http://localhost:3000/products.html

## 專案結構

```
├── products.html       # 商品列表頁
├── product-detail.html # 商品詳情頁
├── cart.html          # 購物車頁面
├── css/style.css      # 樣式
├── js/                # JavaScript
└── images/            # 圖片資源
```

## 內建 UX 痛點

此網站故意埋入了多個 UX 痛點，用於測試 UXAgent 的問題偵測能力：

### 1. 加入購物車通知不足
- **位置**: `css/style.css:1131-1149`, `js/cart.js:110-137`
- **問題**:
  - Toast 通知顏色太淡（灰色 #e8e8e8）
  - 字體太小（11px）、透明度低（0.6）
  - 顯示時間太短（只有 1.5 秒）
  - 使用者很容易錯過加入購物車的反饋

### 2. 商品介紹文字太小
- **位置**: `css/style.css:801-807`
- **問題**:
  - 商品說明段落和列表的字體只有 11px
  - 顏色太淡（#999999）
  - 行距過小（1.4），閱讀困難

### 3. 按鈕顏色對比不足
- **位置**: `css/style.css:671-704`
- **問題**:
  - "加入購物車"按鈕使用淡灰色（#f0f0f0），文字也是灰色（#888888）
  - "立即購買"按鈕顏色太淡（#d4a5a5）
  - "我的收藏"按鈕幾乎看不見
  - 無法清楚辨識哪個是主要操作

### 4. 價格資訊不清楚
- **位置**: `css/style.css:524-553`
- **問題**:
  - 原價和優惠價的對比不明顯
  - 折扣標籤顏色太淡（#e8c8c8）
  - 價格字體顏色都偏灰
  - 優惠期限字體過小（11px）

### 5. 數量按鈕太小
- **位置**: `css/style.css:619-660`
- **問題**:
  - 加減數量按鈕只有 28x28px，難以點擊
  - 數量輸入框也變小到 45x28px
  - 字體縮小到 13-14px
  - 在手機上更難操作

### 6. 搜尋框不清楚
- **位置**: `css/style.css:61-87`
- **問題**:
  - Placeholder 顏色太淡（#d8d8d8）
  - 搜尋按鈕顏色不明顯（#c9a5a5）
  - 按鈕文字顏色太淡（#e8e8e8）

### 7. 導航連結間距不足
- **位置**: `css/style.css:128-152`
- **問題**:
  - 導航連結間距減少到 18px
  - 字體縮小到 13px，顏色偏灰（#666666）
  - 點擊區域過小
  - hover 和 active 狀態不明顯

### 8. 商品列表標題字太小
- **位置**: `css/style.css:390-425`
- **問題**:
  - 商品標題字體只有 12px
  - 顏色偏灰（#888888）
  - 價格顯示不夠突出（18px, #999999）
  - 折扣資訊幾乎看不見（10px, #cccccc）

### 9. 配送選項難以閱讀
- **位置**: `css/style.css:725-748`
- **問題**:
  - 配送方式標題縮小到 14px
  - 選項文字只有 11px，顏色太淡（#999999）
  - 在選擇配送方式時造成困擾

### 10. 視圖切換按鈕不明顯
- **位置**: `css/style.css:272-291`
- **問題**:
  - 按鈕顏色太淡（#fafafa, #f0f0f0）
  - 文字顏色太灰（#aaaaaa, #999999）
  - 使用者不容易發現可以切換視圖

## 測試 UXAgent

```bash
# 啟動網站
python3 -m http.server 3000

# 執行 UXAgent（另開終端機）
cd /Users/akiraeason/Desktop/UXAgent
uv run -m src.simulated_web_agent.main \
  --intent "瀏覽商品並加入購物車" \
  --start-url "http://localhost:3000/products.html" \
  --max-steps 20
```
